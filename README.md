En Apache, hacer un link simbolico a www del proyecto. El nombre del link simbolico debe ser beeter.
En Windows, el comando para realizar esto es:
	mklink /d beeter "Ruta al www" (desde el directorio de Apache/htdocs)
	
La URL es http://localhost:80/beeter/

El resto de la navegación es intuitiva.
